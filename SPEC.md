this is an at-a-glance app for a pro user

hero module with:
- location name
- observation time
- current temperature
- feels like temperature
- current weather condition and icon
- toggle to switch celsius and fahrenheit

compact measurement cluster:
- humidity
- dew point
- pressure
- precipitation rate
- precipitation total for today
- wind speed, direction and gust
- UV index
- dawn / sunrise / sunset / dusk times as a diagram showing the sun at its current position (https://en.wikipedia.org/wiki/Twilight#/media/File:Twilight_description_full_day.svg)


radar module:
- map
- scrubbable timeline (last 2 hours)
- play / stop toggle


5-day forecast module, each day has a card with:
- day of week name
- daytime weather icon
- daytime precipitation percentage
- daytime temperature
- nighttime weather icon
- nighttime precipitation percentage
- nighttime temperature

7-day temperature history module:
- chart with high and low temperature by day, last 7 days
- current temperature shown on today
